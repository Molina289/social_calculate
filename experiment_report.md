
# 金融诈骗检测实验报告

## 1. 数据概况
- 总样本数：5,147
- 诈骗样本比例：53.43%
- 训练集大小：4,117
- 测试集大小：1,030

## 2. 特征工程
### 2.1 文本特征
- 清洗：移除URL、@提及、表情符号等
- TF-IDF：500维，1-2 gram

### 2.2 对话特征（共3个）
1. 对话长度 (dialog_length)
2. 关键词数量 (keyword_count)
3. 提及金额 (amount_mentioned)

## 3. 实验结果
| 模型 | AUC | 精确率 | 召回率 | F1分数 |
|------|-----|--------|--------|--------|
| 仅文本 | 0.9786 | - | - | - |
| 文本+对话特征 | 0.9846 | - | - | - |

**AUC提升：0.0060**

## 4. 结论
对话特征对金融诈骗检测有明显提升效果

## 5. 文件输出
1. `predictions.csv`：测试集预测结果
2. `text_only_model.pkl`：仅文本模型
3. `combined_model.pkl`：文本+对话特征模型
